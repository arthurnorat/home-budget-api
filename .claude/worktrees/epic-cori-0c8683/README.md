# Orçamento

Aplicativo de controle de gastos domésticos compartilhado entre dois usuários.

## O que faz

- Registra gastos com descrição, valor, data e categoria (fixo ou variável)
- Lista os gastos do mês
- Dados compartilhados entre dois usuários em tempo real

## Stack

- **Backend:** Java + Spring Boot + PostgreSQL (Railway)
- **Frontend web:** React (Vercel)
- **Mobile (futuro):** React Native
